Entrega de la practica 2 - Seminario de Lenguajes - Opcion Python

El repositorio contiene la resolucion de los ejercicios de la practica en sus archivos correspondientes .py, dentro de la carpeta src.
Ademas, se incluye un notebook en notebooks/, correspondiente al ejercicio entregable.

Requisitos para instalar el repositorio:
- Python 3
- Jupyter Notebook


Pasos para la instalacion:

1- Clonar el repositorio:
git clone https://github.com/lucaslopez284/Python2026_Entrega_Practica_Dos.git
cd Python2026_Entrega_Practica_Dos

2- Crear entorno virtual:
python -m venv venv

3- Activar entorno virtual: 
En Windows:
venv\Scripts\activate

4- Instalar dependencias:
pip install -r requirements.txt

Pasos para ejecutar el ejercicio entregable:

1- Ejecutar en la terminal:
jupyter notebook

2- Abrir:
notebooks/ejercicio_10.ipynb
